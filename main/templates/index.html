Website defacement
Posted by en.wikipedia.org, January 1, 2025 at 6:45 am, in Web Template.

Website defacement is an attack on a website that changes the visual appearance of a website or a web page.

These are typically the work of hackers, who break into a web server and replace the hosted website with malware or a website of their own.
